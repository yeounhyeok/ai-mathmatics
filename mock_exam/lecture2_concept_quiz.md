## 2. 선형변환 조건과 반례

다음 변환들이 선형변환인지 판단하고 이유를 쓰시오.

```text
T1(x) = A @ x
T2(x) = A @ x + b
T3(x) = 2x
T4(x) = x + 3
```

답안에는 다음 두 조건을 반드시 사용하시오.

```text
가산성: T(x + y) = T(x) + T(y)
동차성: T(cx) = cT(x)
```

추가 질문:  
`T2(x) = A @ x + b`는 일반적으로 선형변환이 아니다. 그런데 왜 편향 트릭을 쓰면 행렬곱 형태로 다룰 수 있는가?

## 4. `np.newaxis`, `reshape`, 차원 추가

다음 코드가 있다.

```python
x = np.array([10, 20, 30])
```

다음 결과 shape를 쓰시오.

```text
(a) x.shape
(b) x[:, np.newaxis].shape
(c) x[np.newaxis, :].shape
(d) x.reshape(-1, 1).shape
(e) x.reshape(1, -1).shape
```

다음 연산의 결과 shape도 쓰시오.

```python
x[:, np.newaxis] + np.array([1, 2])
```

추가 질문:  
`x[:, np.newaxis]`와 `x[np.newaxis, :]`는 둘 다 차원을 하나 추가하지만 의미가 다르다. 각각을 “열벡터”, “행벡터”라는 단어를 사용해 설명하시오.

## 5. 편향 트릭과 코드 빈칸

다음 모델이 있다.

```text
y_hat = 3*x1 - 2*x2 + 7
```

### 5-1. 수식

입력 벡터를 `[x1, x2, 1]`로 확장했을 때 가중치 벡터 `w_aug`를 쓰시오.

```text
w_aug = ?
```

### 5-2. 행렬 전체에 적용

샘플이 4개, 특성이 2개인 행렬 `X`가 있다.

```text
X.shape = (4, 2)
```

편향 트릭을 적용해 `X_aug`를 만들면 shape는 무엇인가?

```text
X_aug.shape = ?
```

### 5-3. 코드 빈칸

다음 코드를 완성하시오.

```python
m = X.shape[0]
ones = np.ones((___, ___))
X_aug = np.hstack([X, ___])
w_aug = np.array([[3], [-2], [7]])
y_hat = X_aug ___ w_aug
```

### 5-4. 개념 설명

편향 트릭을 쓰면 `X @ w + b`를 `X_aug @ w_aug` 하나로 표현할 수 있다.  
이 방식이 수학적으로나 코드 구현상 편한 이유를 설명하시오.
