## 3. 선형회귀, MSE, 경사하강법 / 20점

### 3-1. 예측값, 잔차, MSE / 6점

```text
X = [[1, 1],
     [1, 2],
     [1, 3]]

w = [1, 2]
y = [4, 6, 8]
```

다음을 모두 구하시오.

```text
y_hat = X @ w
residual r = y - y_hat
MSE = (1/m) * sum((y_i - y_hat_i)^2)
```

### 3-2. MSE gradient 유도 / 8점

다음 조건에서 MSE gradient를 유도하시오. chain rule 과정이 드러나야 합니다.

```text
L(w) = (1/m) * ||y - Xw||^2
y_hat = Xw
r = y - y_hat

목표:
grad L = -(2/m) * X.T @ (y - y_hat)
또는
grad L =  (2/m) * X.T @ (y_hat - y)
```

### 3-3. 경사하강법 2스텝 / 4점

```text
w0 = 1
alpha = 0.2

step 1 gradient = -4
step 2 gradient =  1
```

업데이트 규칙 `w <- w - alpha * gradient`를 사용하여 w1, w2를 구하시오.

### 4-4. Sigmoid chain rule 유도 / 6점

다음을 유도하시오.


### 5-2. 이항분포 우도 계산 / 4점

동전을 3번 던져 앞면이 2번 나왔다. `p = 0.5`일 때 우도 `L(p)`를 계산하시오.

```text
L(p) = C(n,k) * p^k * (1-p)^(n-k)
n = 3
k = 2
p = 0.5
```

### 5-3. MLE 유도 / 7점

이항분포에서 다음을 유도하시오.

```text
L(p) = C(n,k) * p^k * (1-p)^(n-k)
ell(p) = log L(p)

목표:
p_hat = k/n
```

힌트: `log C(n,k)`는 p에 대해 상수입니다.
