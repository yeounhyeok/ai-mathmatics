다음을 구하시오.

  

```text

```

  

### 3-2. MSE 계산 / 4점

  

3-1의 결과를 이용하여 MSE를 구하시오.

  

```text

MSE = (1/m) * sum((y_i - y_hat_i)^2)

```

  

### 3-3. MSE gradient 유도 / 8점

  

다음 조건에서 MSE gradient를 유도하시오. chain rule 과정이 드러나야 합니다.

  

```text

L(w) = (1/m) * ||y - Xw||^2

y_hat = Xw

r = y - y_hat

  

목표:

grad L = -(2/m) * X.T @ (y - y_hat)

또는

grad L =  (2/m) * X.T @ (y_hat - y)

```

  

### 3-4. 경사하강법 2스텝 / 5점

  

```text

w0 = 1

alpha = 0.2

  

step 1 gradient = -4

step 2 gradient =  1

```

  

업데이트 규칙 `w <- w - alpha * gradient`를 사용하여 w1, w2를 구하시오.

### 5-3. 이항분포 우도 계산 / 3점

  

동전을 3번 던져 앞면이 2번 나왔다. p = 0.5일 때 우도 L(p)를 계산하시오.

  

```text

L(p) = C(n,k) * p^k * (1-p)^(n-k)

n = 3

k = 2

p = 0.5

```

  

### 5-4. MLE 유도 / 4점

  

이항분포에서 다음을 유도하시오.

  

```text

L(p) = C(n,k) * p^k * (1-p)^(n-k)

ell(p) = log L(p)

  

목표:

p_hat = k/n

```

  

힌트: `log C(n,k)`는 p에 대해 상수입니다.
