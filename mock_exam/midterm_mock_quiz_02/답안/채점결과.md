## 핵심 오답 정리

### 1. 1D transpose

```python
x = np.array([1, 2, 3])
x.shape      # (3,)
x.T.shape    # (3,)
```

1D 배열은 행벡터/열벡터 구분이 없으므로 `.T`를 해도 바뀌지 않는다. Column vector가 필요하면 아래처럼 바꿔야 한다.

```python
x.reshape(-1, 1)
x[:, np.newaxis]
```

### 2. linspace

```python
np.linspace(0, 1, 5)
```

정답:

```text
[0, 0.25, 0.5, 0.75, 1]
```

`linspace(a, b, c)`는 `a`부터 `b`까지 `c`개를 만들고, 끝값 `b`를 포함한다.

### 3. 편향 트릭 부호

```text
y_hat = 2*x1 - 3*x2 + 5
```

입력을 `[x1, x2, 1]`로 확장하면:

```text
w = [2, -3, 5]
```

### 4. Broadcasting

```text
(2, 1) + (1, 3) -> 가능, 결과 shape (2, 3)
```

브로드캐스팅은 뒤 차원부터 비교한다. 둘 중 하나가 1이면 상대 차원으로 확장 가능하다.

### 5. 표준화에서 훈련 통계량 사용

시험 답안:

```text
테스트 데이터의 평균/표준편차를 사용하면 테스트 정보가 학습 과정에 새는 데이터 누수가 생기므로,
훈련 데이터에서 계산한 평균과 표준편차를 사용해야 한다.
```

### 6. 과적합과 Weight Decay

시험 답안:

```text
과적합은 훈련 손실은 낮지만 테스트 손실은 높아지는 상태이다.
```

```text
Weight Decay는 손실함수에 lambda * ||w||^2 항을 추가해 큰 가중치에 벌점을 주는 방법이다.
```

### Lab 2 / Lecture 2

브로드캐스팅 규칙:


```python
ones = np.ones((m, 1))
```

```text
np.ones(m)      -> shape (m,)
```
