## 1. NumPy, Shape, Broadcasting, 인덱싱 / 20점

### 1-1. Shape와 행렬곱 / 5점

다음 연산이 가능한지 O/X로 표시하고, 가능하면 결과 shape를 쓰시오.

```text
(a) X: (5, 3), w: (3,)       -> X @ w
(b) X: (5, 3), w: (3, 1)     -> X @ w
(c) X: (5, 3), v: (5,)       -> X @ v
(d) a: (3,), b: (3,)         -> np.dot(a, b)
(e) a: (3,), b: (3, 1)       -> a @ b
```

### 1-2. Broadcasting / 5점

각 연산이 가능한지 O/X로 표시하고, 가능하면 결과 shape를 쓰시오.

```text
(a) A: (4, 3), b: (3,)       -> A + b
(b) A: (4, 3), c: (4,)       -> A + c
(c) A: (4, 3), d: (4, 1)     -> A + d
(d) u: (3,), v: (3, 1)       -> u + v
(e) M: (2, 3, 4), q: (3, 1)  -> M + q
```

### 1-3. 인덱싱 / 5점

다음 배열이 있다.

```python
X = np.array([
    [10, 20, 30],
    [40, 50, 60],
    [70, 80, 90],
    [11, 22, 33]
])
```

각 코드의 결과 shape를 쓰고, 인덱싱 종류를 쓰시오.

```python
(a) X[1, 2]
(b) X[1:3, :]
(c) X[X[:, 0] > 30]
(d) X[:, [2, 0]]
(e) X[[0, 2], [1, 2]]
```
